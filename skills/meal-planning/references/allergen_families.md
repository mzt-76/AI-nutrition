# Familles d'Allergenes - Filtrage Actif

## Workflow de Securite

1. **AVANT toute suggestion d'aliment** : Verifie le profil pour allergies ET disliked_foods
2. **FILTRE activement** : Exclus TOUS les aliments de la famille allergene declaree
3. **Verifie les memoires** : Cherche "allergique", "allergie", "intolerant" dans le contexte
4. **Suggere des alternatives sures** : Alternatives nutritionnellement equivalentes sans allergenes

## Filtrage par Famille

### Arachides
- INTERDIT : Arachides, beurre de cacahuete, huile d'arachide, sauce satay
- ALTERNATIVES : Graines (tournesol, courge)

### Fruits a coque
- INTERDIT : Amandes, noix, cajou, noisettes, pistaches, noix de pecan, noix de macadamia
- ALTERNATIVES : Graines (chia, lin, courge, tournesol)

### Lactose
- INTERDIT : Lait, yaourt, fromage, creme, beurre
- ALTERNATIVES : Lait d'amande, yaourt soja, lait d'avoine

### Gluten
- INTERDIT : Ble, orge, seigle, epeautre, pain classique, pates classiques
- ALTERNATIVES : Riz, quinoa, sarrasin, mais, pates sans gluten

## Note Importante

- "Noix de coco" n'est PAS un fruit a coque botaniquement (c'est une drupe)
- Les personnes allergiques aux fruits a coque peuvent generalement consommer de la coco
- Utilise le terme "coco" plutot que "noix de coco" pour eviter la confusion
