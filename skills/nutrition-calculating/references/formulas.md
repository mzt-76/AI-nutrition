# Formules et References Scientifiques

## Formule Mifflin-St Jeor (BMR)

**Source** : Mifflin et al. (1990) - "A new predictive equation for resting energy expenditure in healthy individuals"

- **Hommes** : BMR = (10 x poids_kg) + (6.25 x taille_cm) - (5 x age) + 5
- **Femmes** : BMR = (10 x poids_kg) + (6.25 x taille_cm) - (5 x age) - 161

## Multiplicateurs d'activite (TDEE)

| Niveau | Multiplicateur | Description |
|--------|---------------|-------------|
| Sedentaire | 1.2 | Bureau, peu d'exercice |
| Leger | 1.375 | Exercice leger 1-3x/semaine |
| Modere | 1.55 | Exercice modere 3-5x/semaine |
| Actif | 1.725 | Exercice intense 6-7x/semaine |
| Tres actif | 1.9 | Athlete, travail physique intense |

## Ajustements Caloriques par Objectif

| Objectif | Ajustement | Rationale |
|----------|-----------|-----------|
| Perte de poids | -500 kcal/jour | ~0.5kg perte/semaine (defaut) |
| Prise de muscle | +300 kcal/jour | Surplus modere pour minimiser gain de gras |
| Performance | +100 kcal/jour | Leger surplus pour soutenir entrainements |
| Maintenance | 0 | Equilibre energetique |
